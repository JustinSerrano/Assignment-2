# Assignment 2

## Overview
This is a Java-based application designed to manage an inventory of toys for a toy store. It allows users to add, remove, search, and suggest gifts based on criteria such as age, type, and price range. The program provides a command-line interface for user interaction and includes features for ensuring data consistency and validation.

This project is developed by **Group 6**:
- **Justin**
- **Fatema**
- **Manveet**

## Features
- **Inventory Management**: Add, remove, and update toy information.
- **Search Functionality**: Search for toys by serial number, name, or type.
- **Gift Suggestions**: Suggest toys based on optional filters like age, type, and price.
- **Data Persistence**: Save inventory data to a text file and load it on startup.

- ## Prerequisites
- **Java JDK 8** or higher
- **Git** (optional, for cloning the repository locally)
- A command-line interface (e.g., Command Prompt, Terminal) for running the JAR file

- ## Runnable JAR file
- If you'd like to work with the project code locally, you can clone the repository:
- https://github.com/JustinSerrano/Assignment-2
